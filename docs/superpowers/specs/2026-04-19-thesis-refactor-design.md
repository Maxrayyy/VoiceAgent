# 毕业论文重构设计（同济 2026 模板规范）

## 背景

当前 `毕业论文.docx` 存在以下问题：
1. 第二章技术选型缺少"备选方案对比 + 选型理由"；2.3-2.6（STT/TTS/WebSocket/异步）结构散乱，WebSocket 不应单独列一节
2. 第四章缺少"整体实现概述 + 架构图 + 流程图"；文档预处理作为独立模块与运行时模块并列不合理；WebSocket 服务与前端实现不应和 STT/LLM/TTS 等核心功能模块平级
3. 正文字数约 4.3w 字，远超 1.5w-2w 的目标
4. 生成脚本 `scripts/generate_thesis_docx.py` 的格式不完全符合同济 2026 模板气泡规范（字号、编号、对齐、三级以下标题样式、段落层级编号等）

## 目标

- 按同济模板气泡规范重写格式生成脚本
- 重构论文结构使其满足"技术选型可对比、模块划分平行、整体架构清晰"
- 将总字数压缩到 1.5w-2w 字
- 生成架构图 / 流程图 / 用例图保存到 `thesis/figures/`
- 把 md→docx 能力封装为可复用 skill

## 同济 2026 模板格式规范

### 页面设置

- 纸型 A4，页边距：上下 2.5cm，左 3.0cm，右 2.0cm
- 页眉居中"毕业设计（论文）"小五号宋体
- 页脚居中页码（Arabic 数字），起始页从正文开始编号

### 封面

- `TONGJI UNIVERSITY`：二号 TNR 加粗居中
- `毕业设计（论文）`：初号级别黑体加粗居中
- **课题名称：小二号（18pt）黑体 加粗 居中 行距 1.5 倍 段前 0.5 行 段后 0.5 行**（原脚本为三号，需改）
- 题目前后各空一行；有副标题则用三号字体
- 信息项（学院/专业/姓名/学号/指导教师/日期）：四号宋体 居中

### 中文摘要

- 中文课题名：**小二号黑体加粗居中**（原脚本小二 ✓）
- `摘  要`：**四号黑体加粗居中** 段前段后 0.5 行 行距 1.5 倍（原脚本三号，需改）
- 正文：小四宋体 首行缩进 2 汉字 行距 1.5 倍
- 空一行后 `关键词：`：小四宋体加粗顶格
- 关键词：小四宋体，用中文逗号"，"分隔，最后一个关键词后无标点

### 英文摘要（独立一页）

- 英文课题名：**小二号 Times New Roman 加粗居中 首字母大写** 段前段后 0.5 行
- `ABSTRACT`：**四号 TNR 加粗居中** 段前段后 0.5 行
- 正文：小四 TNR 首行缩进 2 汉字符 行距 1.5 倍
- `Keywords: `：小四 TNR 加粗顶格
- 关键词：小四 TNR，用英文逗号 + 空格 ", " 分隔

### 目录

- `目  录`：**小三号黑体居中**（两字中间空一格）段前段后 0.5 行 行距 1.5 倍
- 目录条目：小四宋体（英文 TNR）单倍行距
- 使用 Word 域自动生成

### 正文章节标题

| 级别 | 字号 | 字体 | 对齐 | 缩进 | 段前段后 | 其他 |
|------|------|------|------|------|----------|------|
| 章（1级） | **小三号(15pt)** | 黑体 | 居中 | — | 0.5 行 | **每章换页 + 空一行** 行距 1.5 倍 |
| 节（2级） | **小四号(12pt)** | 黑体 | 顶格 | — | 0.5 行 | 行距 1.5 倍 |
| 小节（3级） | 小四号 | 黑体 | — | **缩进 2 汉字** | 0.5 行 | 序号与题名空一格 |
| 三级以下 | 小四号 | 黑体 | — | **缩进 2 汉字** | 0.5 行 | 使用 A./B./C. 或 a./b./c. 或（1）（2）（3）或 ①②③ |

**注意**：章标题字号从三号(16pt)改为小三(15pt)，节标题从四号(14pt)改为小四(12pt)。

### 正文

- 字号：小四号
- 字体：宋体（英文 Times New Roman）
- **对齐：两端对齐**
- 首行缩进 2 汉字（约 0.74cm）
- 行距：1.5 倍
- 段前段后 0 行

### 段落层级编号

- 段落号：**`（1）（2）（3）`** 用全角括号
- 段内层次号：**`①②③`**

### 表格

- **表序命名：表 X.Y（用点号），非 X-Y**（原脚本使用 X-Y，需改）
- 表题：小五号宋体（英文 TNR）加粗居中 置于**表上方** 行距 1.5 倍
- 表序与表题之间空一格，表题末尾无标点
- 表格上下与正文各空一行
- 三线表：两端与页面对齐，上下粗线 + 表头下细线
- 表中文字：小五号宋体 行距 18 磅 段前段后 0 行
- 表格分页：第 2 页省略表题，重写表头，表右上方加"续表 X.Y"（小五号）

### 图

- 图序命名：图 X.Y（按章编号）
- 图：居中，上下与正文各空一行
- 图题：小五号宋体 加粗 居中 置于**图下方** 行距 1.5 倍
- 图序与图题之间空一格
- 图中文字：小五号宋体 行距 1 倍

### 公式

- 另起一行，居中显示
- 编号：**`(X.Y)` 圆括号 章节编号**，标注于行末 右对齐
- 公式行距 1.5 倍
- 物理量/变量用斜体，元素符号/记号/运算符号用正体
- 同一行只允许出现一次 `/`

### 参考文献

- `参考文献`：**四号黑体居中**（原脚本三号，需改）
- 条目：`[N] 作者. 题名. 来源...` 小四宋体（英文 TNR）行距 1.5 倍 顶格 悬挂缩进
- 不少于 10 篇，外文不少于 2 篇
- 按在正文中出现次序用阿拉伯数字自然编号

### 致谢

- `致  谢`（或`谢  辞`）：**四号黑体居中**（原脚本三号，需改）
- 正文：小四宋体 首行缩进 2 汉字 两端对齐 行距 1.5 倍

## 脚本改动清单（Task 1）

编辑 `scripts/generate_thesis_docx.py`：

1. 字号常量保持（小三=15pt 已定义）
2. `add_chapter_heading`：SAN_HAO → **XIAO_SAN**；换页 + 空一行（标题前添加空段落）
3. `add_section_heading`：SI_HAO → **XIAO_SI**；顶格（first_indent=0）
4. `add_subsection_heading`：缩进 2 汉字 ✓；段前 0.5 行
5. `add_body_paragraph`：增加 `alignment=WD_ALIGN_PARAGRAPH.JUSTIFY`
6. 表编号：`detect_table_caption` 中 `表\s*[\d]+-[\d]+` → `表\s*[\d]+[\.-][\d]+`，并把源 Markdown 里 `表 X-Y` 批量改为 `表 X.Y`
7. 图片插入：新增 `add_figure(doc, img_path, caption, chapter_num)` 函数，识别 Markdown `![caption](path)` 语法
8. 公式编号：新增 `add_formula(doc, text, number)`，支持尾部 `(X.Y)` 右对齐
9. 中文摘要标题 `摘  要`：SAN_HAO → SI_HAO
10. 英文摘要 `ABSTRACT`：SAN_HAO → SI_HAO
11. 目录 `目  录`：SAN_HAO → **XIAO_SAN**
12. `add_references` / `add_acknowledgment` 的章标题：SAN_HAO → SI_HAO 且**不换页但需换页**，修正为换页+四号黑体
13. 识别 Markdown 里的 `三级以下单占行标题`（检测 `(1) xxx`、`① xxx` 在行首且独占一段时应用标题样式）— 仅在 `**(1) xxx**` 加粗语法时识别为三级以下标题
14. 正文列表项 `（1）（2）（3）` 和 `①②③` 保持正文样式（不是标题，正文内分项）

## Skill 封装（Task 7）

创建 `.claude/skills/md-to-thesis-docx/`：

```
.claude/skills/md-to-thesis-docx/
  SKILL.md              # skill 描述（触发条件：用户要求把 thesis/*.md 转为 docx 或修改论文格式）
  convert.py            # 封装后的转换脚本（从 generate_thesis_docx.py 迁移）
  format_spec.md        # 同济 2026 模板格式规范（本 spec 的格式章节单独抽出）
  README.md             # 使用说明
```

SKILL.md 描述：当用户请求将 `thesis/` 下的 Markdown 文件转换为同济大学毕业设计 docx 论文时调用，支持封面、中英文摘要、目录、七章正文、参考文献、致谢的完整生成。
