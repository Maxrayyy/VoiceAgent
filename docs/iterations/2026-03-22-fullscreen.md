# 全屏极简语音界面改造计划

## 目标
- 页面占满视口，去除一切卡片/边框，营造浅米白极简风。
- 进入页面即尝试自动开始录音，取消“按住”交互。
- 顶部中央仅保留实时波形；左上角展示产品名。
- 右侧实现可折叠侧边栏，默认仅剩窄触发条，展开后显示系统状态与操作，不出现对话内容。
- 在用户发声后，按出现顺序以悬浮文本逐行展示识别与回复，常态空白。

## 代码变更范围
1. `src/server/static/index.html`
   - 改造基础结构：`<body>` 只包含品牌标签、波形容器、转写堆栈和侧边栏。
   - 移除原有聊天区/按钮/来源弹窗，新增折叠侧栏内部操作按钮（打断、清除、查看来源）。
2. `src/server/static/style.css`
   - 重设配色为浅米白背景 + 极简排版；波形、文字均无边框。
   - 定义全屏布局、悬浮文字动画、侧边栏滑出效果与触发条样式。
3. `src/server/static/app.js`
   - 移除文本输入/历史逻辑，保留音频播放与 RAG 来源处理。
   - DOMContentLoaded 时自动调用 `startRecording()`；处理用户拒绝权限时的提示。
   - 实现波形绘制（canvas 无背景）、悬浮转写堆栈添加/淡出、侧栏展开收起逻辑。
   - 侧栏按钮承载 `interrupt`、`clearHistory`、`showSources` 等现有功能。
4. 其余：保留 `FRONTEND_PLAN.md` 作为上一版记录，不再修改。

## 验证清单
- 页面加载后自动弹出麦克风权限，允许后波形滚动、无 UI 抖动。
- 未说话时除左上角文本、顶部波形、右侧触发条外保持空白。
- 说话后可连续看到 STT 文本行和 AI 回复行，且它们悬浮无边框。
- 侧栏默认收起，鼠标悬停/点击可展开，包含打断、清除、参考来源列表。
- 参考来源在侧栏中更新，无与对话内容重叠。
